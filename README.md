# AI Autocorrect Tool 🤎✨

An AI-powered web application that automatically corrects spelling and grammar mistakes in user-entered text using Transformer-based Natural Language Processing (NLP) models.

## 🚀 Features

- AI-powered spelling correction
- Grammar improvement using Transformer models
- Modern brown aesthetic user interface
- Original vs Corrected Text comparison
- Character Count and Word Count
- Copy Corrected Text functionality
- FastAPI backend
- Responsive web design

---

## 🛠️ Technologies Used

- Python
- FastAPI
- HappyTransformer
- Transformers
- HTML
- CSS
- Jinja2

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Autocorrect-Tool.git
cd AI-Autocorrect-Tool
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
uvicorn app:app --reload
```

Open:

```text
http://127.0.0.1:8000
```
